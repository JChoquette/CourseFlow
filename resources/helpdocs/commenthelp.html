Comments

Comments can be added to a workflow to leave notes when sharing the workflow.

Getting Started

To add a comment to a workflow, right click anywhere in the workflow and click "Add Comment". This will add a comment to the workflow. You can see the comment by clicking on it.

Manipulating Comments

  • Moving Comments
    • Comments can be moved by clicking and dragging them.
  • Adding to Comments
    • You can add to a comment by clicking it to display it, then typing text and clicking "add comment". The comment will persist when the workflow is shared, and this can be a good way to share feedback with others.
  • Deleting Comments
    • Comments can be deleted by clicking the icon that appears when you hover over them.
